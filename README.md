# excel-to-json-in-Node.js

##Quick Setup

1) `clone the branch`  <br>
2) `cd excel-to-json-in-Node.js` <br>
3) `npm install` <br>
4) `node app.js` <br>
5) In your browser `http://localhost:3000` <br>
6) Upload excel file and see result <br>
